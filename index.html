About me

Halo, saya Elang, seorang mahasiswa teknik informatika di PENS yang sangat antusias tentang machine learning, pengembangan web, dan pemodelan 3D. Saya sangat senang menghadapi tantangan dalam menerapkan algoritma machine learning untuk masalah dunia nyata dan menikmati membuat situs web interaktif menggunakan teknologi terkini di bagian depan dan belakang. Selain itu, saat ini saya sedang menjelajahi dunia pemodelan 3D untuk menciptakan pengalaman mendalam untuk metaverse. Tujuan saya adalah memanfaatkan keterampilan saya dan berkontribusi pada kemajuan teknologi, berkolaborasi dengan orang lain untuk membentuk masa depan yang lebih baik.
